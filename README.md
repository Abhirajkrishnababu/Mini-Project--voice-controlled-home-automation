# Simple Voice Demo Home Automation (No Authentication)

This is a simplified software-only home automation prototype.  
It does **not** use ML or authentication — it only records a 10-second demo voice to show UI interaction.

## Features

✔ Record admin voice (demo only)  
✔ After recording → home automation UI unlocks  
✔ Toggle Bulb  
✔ Toggle Fan  
✔ Happy Diwali Mode (rainbow glowing border)  
✔ Reset system (clears audio + resets UI)  
✔ Very simple and perfect for viva / GitHub  

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI (Python)
- Audio: MediaRecorder API + Pydub
